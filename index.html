关键字的解释与举例

  • 一、break

    break 语句用于中止当前循环或 switch 语句或 label 语句，并把程序控制流转到紧接着被中止语句后面的语句。

    例句1:循环语句

    for(var i=0;i<10;i++){
      if(i==3) break;
      console.log(i);　　//0,1,2
    }

    例句2:switch语句

    function main(arg){
      switch(arg){
        case 1:console.log(1); break;
        case 2:console.log(2);
      }
      console.log(3);
    }
    main(1);　　//1,3

    例句3:lable语句

    var num = 0;
    outPoint:
    for (var i = 0 ; i < 10 ; i++){
      for (var j = 0 ; j < 10 ; j++){
        if( i == 5 && j == 5 ){
          break outPoint;
        }
      num++;
      }
    }
    alert(num); // 循环在 i 为5，j 为5的时候跳出双循环，返回到outPoint层继续执行，输出 55

  • 二、do

    关键字do是用来标识do...while循环的开始，用while来标识循环的结尾并进入循环条件判断，和while循环不同，do...while循环是用分号结尾的，且该循环至少会执行一次。

    例句:

    var num = 6;
    do{
      console.log(num);　　//6
      num++;
    }
    while(num<5);

  • 三、instanceof

    instanceof运算符用来判断一个构造函数的prototype属性所指向的对象是否存在另外一个要检测对象的原型链上。

    语法:

    //obj:实例化对象
    //Object:构造函数
    obj instanceof Object;

    例句1:instanceof的普通的用法

    function Person(){};
    function Student(){};
    var p =new Person();
    var s=new Student();
    console.log(p instanceof Person);//true
    console.log(s instanceof Person);//false

    例句2:继承中判断实例化对象是否属于它的父类

    function Person(){};
    function Student(){};
    var p =new Person();
    Student.prototype=p;//继承原型
    var s=new Student();
    console.log(s instanceof Student);//true
    console.log(s instanceof Person);//true

    例句3:复杂用法

    function Person() {}
    //Object.prototype.__proto__ === null
    console.log(Object instanceof Object); //true
    //第一个Object的原型链：Object=>Object.__proto__ => Function.prototype=>Function.prototype.__proto__=>Object.prototype
    //第二个Object的原型：Object=> Object.prototype

    console.log(Function instanceof Function); //true
    //第一个Function的原型链：Function=>Function.__proto__ => Function.prototype
    //第二个Function的原型：Function=>Function.prototype

    console.log(Function instanceof Object); //true
    //Function的原型链：Function=>Function.__proto__=>Function.prototype=>Function.prototype.__proto__=>Object.prototype
    //Object的原型：Object => Object.prototype

    console.log(Person instanceof Function); //true
    //Person的原型链：Person=>Person.__proto__=>Function.prototype
    //Function的原型：Function=>Function.prototype

    console.log(String instanceof String); //false
    //第一个String的原型链：String=>String.__proto__=>Function.prototype=>Function.prototype.__proto__=>Object.prototype
    //第二个String的原型链：String=>String.prototype

    console.log(Boolean instanceof Boolean); //false
    //第一个Boolean的原型链：Boolean=>Boolean.__proto__=>Function.prototype=>Function.prototype.__proto__=>Object.prototype
    //第二个Boolean的原型链：Boolean=>Boolean.prototype

    console.log(Person instanceof Person); //false
    //第一个Person的原型链：Person=>Person.__proto__=>Function.prototype=>Function.prototype.__proto__=>Object.prototype
    //第二个Person的原型链：Person=>Person.prototype

    例句4:需注意的点

    function Person(){};
    var p =new Person();
    console.log(p instanceof Person);//true 改变前
    Person.prototype={};//改变Person.prototype属性的值
    console.log(p instanceof Person);//false 改变后

    函数模拟:

    //A instanceof B
    function _instanceof(A, B) {
      var O = B.prototype;// 取B的显示原型
      A = A.__proto__;// 取A的隐式原型
      while (true) {
      //Object.prototype.__proto__ === null
        if (A === null)
          return false;
        if (O === A)// 这里重点：当 O 严格等于 A 时，返回 true
          return true;
        A = A.__proto__;
      }
    }

  • 四、typeof

    typeof在js中是一个一元操作符，可以判断操作数的类型，其返回值为字符串，有number、string、object、boolean、function、undefined、symbol (ECMAScript 6 新增)，使用方式可以是typeof 操作数或typeof(操作数)。

    例句:

    typeof NaN ; //返回值：number NaN特殊的非数字值
    typeof Infinity ; //返回值：number Infinity无穷大特殊值
    typeof 123 ; //返回值：number
    typeof '123' ; //返回值：string
    typeof(true) ; //返回值：boolean
    typeof(Date) ; //返回值：function
    typeof(s) ; //返回值：undefined 因为s作为一个变量未被定义，所以这里的返回值是undefined
    typeof(undefined) ; //返回值：undefined
    typeof(document) ; //返回值：object document是DOM下的一个对象
    typeof(null) ; //返回值：object
    //Symbol (ECMAScript 6 新增)
    typeof Symbol() ; //返回值：symbol
    typeof Symbol('foo') ; //返回值：symbol
    typeof Symbol.iterator ; //返回值：symbol

  • 五、switch

    switch 语句用于基于不同的条件来执行不同的动作。如果在函数中使用switch语句，可以使用return来代替break，return和break都用于终止switch语句，也会防止一个case语句块执行完后继续执行下一个case语句块。

    例句1:

    switch(v){
    //计算switch小括号里面的表达式，得出结果与case后面的值比较如果相等，则执行case，否则跳过这个case。
      case 1:
      console.log('1');
      break;
      case 2:
      console.log('2');
      break;
      case 3:
      console.log('3');
      break;
    }

    例句2:

    function convert(x){
      switch(typeof x){
        case'number'://将数字转换为十六进制数
        return x.toString(16);
        case'string'://返回两端带双引号的字符串
        return'"'+x+'"';
        default://使用普通的方法转换其他类型
        return String(x);
      }
    }

  • 六、case

    每个case的匹配操作实际上是“===”恒等运算符比较，ECMAScript标准允许每个case关键字跟随任意的表达式。但是由于每次执行switch语句的时候，并不是所有的case表达式都能执行到，因此，应当避免使用带有副作用的case表达式，比如函数调用表达式和赋值表达式。最安全的做法就是在case表达式中使用常量表达式。

    例句:

    var s=3;
    switch(v){//计算v的值
      case 2://计算常量
      console.log('2 true');
      break;
      case 1 + 2://计算数字相加
      console.log('1 + 1 true');
      break;
      case 2* 2://计算数字相乘
      console.log('1 * 2 true');
      break;
      case s * 2://计算变量
      console.log('s * 2 true');
      break;
    }

  • 七、default

    在switch语句中default关键词用来规定匹配不存在时做的事情，"default:"标签可以放置在switch语句内的任何地方。

    例句:

    switch(n){
      case 1:
      console.log(1);
      break;
      case 2:
      console.log(2);
      break;
      case 3:
      console.log(3);
      break;
      default://如果所有的条件都不匹配执行
      console.log("default");
      break;
    }

  • 八、if

    if语句是一种基本的控制语句,有条件地执行语句。

    例句1:

    if(expression)//计算expression的值，如果是true，执行statement。如果是false，不执行statement。
      statement;//一行代码

    例句2:

    if(!address){
      address="";
      message="Please specify a mailing address.";
    }//多行代码

  • 九、else

    if语句的第二种形式引入了else从句,当if的条件值是false的时候执行else中的逻辑。

    例句1:

    if (time<20)
      console.log("Good day");
    else
      console.log("Good evening");

    例句2:

    if (time<10){
      console.log("早上好");
    } else if (time>=10 && time<16){
      console.log("今天好");
    } else{
      console.log("晚上好!");
    }

  • 十、new

    new 运算符创建一个用户定义的对象类型的实例或具有构造函数的内置对象类型之一。

    语法:

    new Constructor(arguments);//Constructor构造函数,传参(arguments)

    例句1:

    function Test(str) {
      this.a = str;
    }
    var myTest = new Test('myTest');//用户自定义的对象
    console.log(myTest);

    例句2:

    var date = new Date();//内置对象
    console.log(date);

  • 十一、var

    变量语句声明了一个变量，可选地将其初始化为一个值。变量声明无论出现在代码的任何位置，都会在任何代码执行之前处理(预解析)。

    例句1:

    var a;//声明一个变量
    var a=1;//声明一个变量并赋值
    var b=[];//声明一个数组
    var c={};//声明一个对象
    var x = y = 1; // x是全局变量，y是隐式全局变量

    例句2:

    // 声明变量的作用域限制在其声明位置的上下文中，而非声明变量总是全局的。
    function x() {
      y = 1; // 在严格模式（strict mode）下会抛出ReferenceError异常。
      var z = 2;
    }
    x();
    console.log(y); // 打印"1" 。
    console.log(z); // 抛出ReferenceError: z未在x外部声明。

    例句3:

    //由于变量声明（以及其他声明）总是在任意代码执行之前处理的，所以在代码中的任意位置声明变量总是等效于在代码开头声明。
    bla = 2;
    var bla;

    // 可以理解为：

    var bla;
    bla = 2;

  • 十二、return

    return语句只能在函数体内出现，如果不是的话会报语法错误。函数中的return语句既是指定函数调用后的返回值,位于return语句之后的任何代码都永远不会执行。

    例句1:

    //没有return语句，调用表达式的结果是undefined。
    function square(){
      var a=1;
    }
    console.log(square());//undefined

    例句2:

    //return语句单独使用返回undefined。
    function square(){
      var a=1;
      return;
    }
    console.log(square());//undefined

    例句3:

    //函数执行到return语句的时候，终止执行，并返回值给调用程序。
    function square(){
      var a=1;
      return a;
    }
    console.log(square());//1

    例句4:

    function square(){
      var a=1;
      var b=1;
      return a,b;//只会返回a的值
    }
    console.log(square());//1

    var arr=[1,2,3,4,5];
    for (var i=0;i<arr.length;i++){
      console.log(arr[i]);

    例句2:

    //for...in遍历对象
    var person={fname:"John",lname:"Doe",age:25};
    for (x in person){
      console.log(person[x]);
    }

  • 十四、while

    例句1:

    例句2:

    var i=0;

  • 十五、void

    void 运算符对任何值返回 undefined。该运算符通常用于避免输出不应该输出的值。还用于填充<a>的href确保点击时不会产生页面跳转; 填充<image>的src，确保不会向服务器发出垃圾请求。

    例句1:

    function x() {
      var undefined = 'hello world',//因为undefined在javascript中不是保留字，可以用undefined做为变量名
      f = {},
      window = {
        'undefined': 'joke'
      };
      console.log(undefined);// hello world
      console.log(window.undefined); //joke
      console.log(f.a === undefined); //false
      console.log(f.a === void 0); //true
    }
    x();

    例句2:

    //使 window.open() 调用返回 undefined，它不是有效值，不会显示在浏览器窗口中。
    <a href="javascript:void(window.open('about:blank'))">Click me</a>
    <img src='javascript:void(0)'/>//确保不会向服务器发出垃圾请求。

  • 十六、continue

    continue语句和break语句相似。所不同的是，它不是退出一个循环，而是转而执行下一次循环。continue语句只能用在while语句、do/while语句、for语句、或者for/in语句的循环体内，在其它地方使用都会引起错误！

    例句:

    for(var i=1;i<=10;i++) {
      if(i==8) {
         continue;//跳出本次循环,开始下次循环
      }
      console.log(i);
    }

  • 十七、debugger

    debugger是谷歌浏览器提供的调试语句，用于停止执行 JavaScript，并调用 (如果可用) 调试函数。使用 debugger 语句类似于在代码中设置断点。需要带有条件的断点，你只需要用if语句包围它.

    例句:

    for(i = 1; i<5; i++) {
      // Print i to the Output window
      console.log("loop index is " + i);
      // Wait for user to resume
      debugger;
    }

  • 二十、with

    with 语句可以方便地用来引用某个特定对象中已有的属性，但是不能用来给对象添加属性。
    注意:
    1、严格模式下不允许使用with语句，否则将视为语法错误。
    2、由于大量使用with语句会到时性能下降，同时也会给调试代码造成困难，因此在开发大型应用程序时，不建议使用with语句。

    例句:

    with(location) {
      var url = href;//可以直接访问location对象中的href属性
    }

  • 二十一、throw

    throw 语句用来抛出一个用户自定义的异常。当前函数的执行将被停止（throw之后的语句将不会执行），并且控制将被传递到调用堆栈中的第一个 catch 块。如果调用者函数中没有catch块，程序将会终止。

    例句1:

    throw '错误';//代码执行至此,抛出错误,后续代码不执行
    console.log(1);

    例句2:

    try {
      throw '错误';//错误被catch捕获,代码可以执行下去
    }catch(e){
      console.log(e);
    }
    console.log('这是最后输出的');

  • 二十二、delete

    delete 操作符会从某个对象上移除指定属性。成功删除的时候回返回 true，否则返回 false。但是，重要的是要考虑以下情况：
    1、如果你删除的属性在对象上不存在，那么delete将不会起作用，但仍会返回true
    2、如果 delete 操作符删除成功，则被删除的属性将从所属的对象上彻底消失。然后，如果该对象的原型链上有一个同名属性，则该对象会从原型链上继承该同名属性。（也就是说delete操作只会在自身的属性上起作用）
    3、任何使用 var 声明的属性不能从全局作用域或函数的作用域中删除。这样的话，delete操作不能删除任何在全局作用域中的函数（无论这个函数是来自于函数声明或函数表达式）
    4、在对象(object)中的函数是能够用delete操作删除的。
    5、任何用let或const声明的属性不能够从它被声明的作用域中删除。(ECMA6中let是用来声明块级作用变量,const用来声明常量)
    6、不可设置的(Non-configurable)属性不能被移除。这意味着像Math, Array, Object内置对象的属性以及使用Object.defineProperty()方法设置为不可设置的属性不能被删除。

    例句1:

    var a=1;
    b=1;
    console.log(delete a;);//false var声明的变量不能被删除
    console.log(delete b);//true 隐式全局变量能被删除

    例句2:

    var Employee = {
      age: 28,
      name: 'abc',
      designation: 'developer'
    }
    console.log(delete Employee.name); // true
    console.log(delete Employee.age); // true
    // 当试着删除一个不存在的属性时
    // 同样会返回true
    console.log(delete Employee.salary); // true

    例句3:

    var arr=[1,2,3,4,5];
    delete arr[1];
    console.log(arr); //删除数组下标 下标会不连续

  • 二十三、in

    如果指定的属性存在于指定的对象中，则 in 运算符会返回 true。in右操作数必须是一个对象值。比如，可以是一个String包装对象，但不能是一个字符串原始值。

    例句:

    var arr = ['red','green','blue'];
    console.log(1 in arr);//true
    var color1 = new String("green");
    "length" in color1 // 返回true
    var color2 = "coral";
    "length" in color2 // 报错(color2不是对象)

  • 二十四、try

    将可能引发错误的代码放在try块中,try语句包含了由一个或者多个语句组成的try块,且至少catch和finally二者之一组成完整的语句，或者两个兼有，如果是语法错误不能被try..catch捕获。

    例句:

    try{
      document.querySelector(".box").style.background='pink';//这里写可能出现异常的代码
    }catch(err){
      console(err);
    }

  • 二十五、catch

    当try语句抛出错误时,执行catch中的语句,catch(e)中的e用来保存被抛出声明指定的值。你可以用这个标识符来获取关于被抛出异常的信息。

    例句:

    try{
      throw '错误';
    } catch(e) {
      console.log(e);//e的值就是抛出的错误内容：'错误'
    }

  • 二十六、finally

    finally 语句在 try 和 catch 之后执行，无论有无异常都会执行。通常用于释放资源。

    try {
      throw '错误';
    }catch(e){
      console.log(e);
    }finally {
      console.log('这是最后输出的');
    }
